4.Introduction Web Programming

Lab HTML

Tag : แท็ก Attribute : คุณลักษณะ Description : อธิบายการใช้งาน
h Align จัดตำแหน่ง กำนดหัวเรื่อง ตั้งแต่ 1-6 (1 ใหญ่สุด และ 6 น้อยสุด)
p Align จัดตำแหน่ง กำนดย่อหน้า
div Align จัดตำแหน่ง เป็นคอนเทนเนอร์เปล่าที่ใช้เพื่อกำหนดส่วน
Img align เพื่อจัดตำแหน่ง
alt กำหนดข้อความสำรองสำหรับรูปภาพ
border กำหนดความกว้างของเส้นขอบรอบ ๆ รูปภาพ
crossorigin เพื่ออนุญาติหรือไม่อนุญาติให้ผู้ใช้ดาว์นโหลดรูปภาพ
height ระบุส่วนสูงของรูปภาพ
hspace กำหนดช่องว่างด้านข้างของรูปภาพ
ismap กำหนดรูปภาพด้าน server side
longdesc แสดง URL เพื่อแสดงคำอธิบายรูปภาพ
referrerpolicy ระบุผู้อ้างอิงที่จะใช้เวลาดึงภาพ
sizes กำหนดขนาดของรูปภาพสำหรับpage layoutที่ขนาดต่างกัน
src แสดง URL ของรูปภาพนั้นๆ
srcset แสดง URL ของรูปภาพนั้นๆเพื่อนำไปใช้ในจุดประสงค์ต่างๆ
usemap กำหนดรูปภาพด้าน client side
vspace กำหนดช่องว่างด้านบนและล่างของรูปภาพ
width ระบุความกว้างของรูปภาพ
 ใช้สำหรับแทรกรูปภาพในเอกสาร HTML
