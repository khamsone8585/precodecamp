Differentiation Tag

Tag : แท็ก similarity : ความเหมือน difference : ความแตกต่าง
hr & br เป็นการแบ่งหรือแยกเนื้อหาข้อมูลออกเหมือนกัน hr แบ่งโดยใช้เส้น
br แบ่งโดยการเพิ่มเว้นหนึ่งบรรทัด
ul & ol จัดข้อมูลเป็นแบบลีสเหมือนกัน ul คือ unordered list ซึ่งจะเรียงในแบบของจุด
ol คือ ordered list ซึ่งจะเรียงโดยใช้ตัวเล็ก
select & checkbox & radio button เป็นแบบฟอร์มของตัวเลือกเหมือนกัน selectรูปแบบของ dropdown lists
checkbox รูปแบบของการกดเลือกในกล่องสี่เหลี่ยมซึ่งสามารถเลือกได้หนื่ง หรีอ มากกว่าหนึ่งช่อง
radio button เป็นรูปแบบของการกดเลือกในกล่องสี่เหลี่ยมได้เพียงกล่องเดียว
div & span เป็น Element เหมือนกัน div เป็น block element จะถูกจัดเรียงกันลงมา
span เป็น inline element จะถูกจัดเรียงติดกันโดยอยุ่ในบรรทัดเดียวกัน
script & style เป็น internal tag ที่ใช้ในไฟล์ html เหมือนกัน script เป็นแท็กสำหรับการเขียน ภาษาjavascript
style เป็นแท็กสำหรับการตกแต่งกับ css

radio สร้างปุ่มตัวเลือก (เมื่อเลือกปุ่มตัวเลือกอื่น ๆ ทั้งหมดจะถูกปิดใช้งาน)
